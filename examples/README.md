# Example Scripts

This directory contains example scripts to help you get started with the tall buildings database.

## Available Examples

### `load_data.py`
Basic Python script demonstrating how to:
- Load the database from CSV
- Explore the data structure
- Generate exploratory visualizations
- Extract key statistics

**Usage:**
```bash
cd examples
python load_data.py
```

**Requirements:**
Install dependencies first:
```bash
pip install -r ../requirements.txt
```
