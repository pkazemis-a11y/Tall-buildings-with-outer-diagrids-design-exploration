# Contributing to Tall Buildings with Outer Diagrids

Thank you for your interest in contributing to this research project! This repository contains research data and visualizations supporting academic work on parametric design of tall buildings with outer diagrids.

## Ways to Contribute

### 1. Report Issues
If you find errors in the data, documentation, or visualizations:
- Open a GitHub issue with a clear description
- Include specific details (e.g., row numbers, variable names)
- Provide steps to reproduce the issue if applicable

### 2. Suggest Improvements
We welcome suggestions for:
- Additional visualizations or analyses
- Documentation enhancements
- Data organization improvements
- New use cases or applications

### 3. Share Your Research
If you use this dataset in your research:
- Let us know by opening an issue or contacting the authors
- We'd love to hear about your findings and applications
- Consider citing our work in your publications

### 4. Extend the Database
Proposals for extending the database are welcome:
- Additional ground motion records
- New geometric parameters
- Different loading conditions
- Alternative structural systems

## Guidelines

### Data Integrity
- Do not modify the original dataset (`data/database.csv`)
- Any derived data should be clearly labeled and documented
- Maintain data provenance and traceability

### Documentation
- Keep documentation clear and concise
- Use proper markdown formatting
- Include references where appropriate
- Update the CHANGELOG if applicable

### Code and Visualizations
If contributing code or visualizations:
- Ensure reproducibility
- Document dependencies and requirements
- Use clear variable names and comments
- Follow existing code style
